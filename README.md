# IBTF-Analytics
Você disse: IBTF RoboAnalytics é uma plataforma desenvolvida para analisar o desempenho dos robôs da equipe IBTF. O sistema organiza dados, identifica problemas, calcula métricas de eficiência e gera relatórios automáticos diretamente para Excel, facilitando o acompanhamento da evolução e a tomada de decisões.
